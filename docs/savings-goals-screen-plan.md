# План екрану відстеження цілей відкладань

## Розташування
- **Вкладка**: Статистика (`/stats`)
- **Підвкладка/секція**: "Відкладання" або "Цілі"

## Функціональність

### 1. Список цілей відкладань
- Відображення всіх категорій типу `savings`
- Для кожної категорії показувати:
  - Назва категорії + іконка
  - Поточна сума відкладень (сума всіх транзакцій типу `savings` з цією категорією)
  - Цільова сума (якщо встановлена в категорії)
  - Прогрес (кольорове кільце/прогрес-бар)
  - Відсоток виконання

### 2. Додавання/редагування цілей
- Можливість встановити цільову суму для категорії відкладань
- Можливість редагувати ціль
- Можливість видалити ціль

### 3. Візуалізація
- Кругові індикатори прогресу (як LimitRing, але для цілей)
- Кольори залежно від прогресу:
  - Зелений: 80-100%
  - Помаранчевий: 50-80%
  - Червоний: <50%

### 4. Фільтрація та сортування
- Фільтр по статусу (активні, досягнуті, всі)
- Сортування по:
  - Найближчій до досягнення
  - Найбільшій сумі
  - Найменшій сумі
  - Назві

### 5. Детальна інформація
- При кліку на ціль - відкрити модальне вікно з:
  - Списком всіх транзакцій відкладань для цієї категорії
  - Графіком прогресу (якщо є історія)
  - Можливістю редагувати ціль

## Структура даних

### Розширення Category для відкладань
```typescript
interface Category {
  // ... існуючі поля
  type: 'savings';
  savingsGoal?: number; // Цільова сума для відкладань
  savingsDeadline?: string; // Дедлайн (опціонально)
}
```

## UI/UX

### Мобільний дизайн
- Список карток з цілями
- Кожна картка - клікабельна, відкриває деталі
- Swipe gestures для швидких дій (видалити, редагувати)

### Desktop дизайн
- Сітка з картками цілей
- Більш детальна інформація на картках
- Hover ефекти

## Інтеграція з існуючим функціоналом
- Використовувати існуючі транзакції типу `savings`
- Використовувати категорії типу `savings`
- Інтегрувати з планом (можливо показувати заплановані відкладання)

## Майбутні покращення
- Графіки динаміки відкладань
- Нагадування про цілі
- Автоматичні відкладання (якщо додати функціонал)
- Експорт даних про відкладання
